Install SID Extension

Takes about 30 seconds. Pick your browser below.

Download SID Extension
Google Chrome
Microsoft Edge
1
Download & unzip the file above. You should see a folder called SID_Extension (or similar) with files inside it.
On Mac, double-click the ZIP to unzip. On Windows, right-click → "Extract All."
2
Open Chrome and go to chrome://extensions
Type or paste chrome://extensions into the address bar and press Enter.
3
Turn on Developer mode using the toggle in the top-right corner of the page.
4
Click "Load unpacked" in the top-left corner.
5
Select the unzipped SID folder (the folder with the files inside, not the ZIP itself) and click "Select Folder."
You should see SID appear in your extensions list with the SID icon.
6
Done! Go to Indeed and SID will be running. You can close the extensions tab.
You may see a small banner about developer mode extensions — just dismiss it.

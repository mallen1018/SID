Install SID Extension

Works with Google Chrome and Microsoft Edge

Download SID Extension
Setup Instructions
1
Click the download button above to save the SID_Extension.zip file.
2
Unzip the file. You should see a folder with the extension files inside.
3
Open your browser and go to chrome://extensions (Chrome) or edge://extensions (Edge).
4
Turn on Developer mode using the toggle in the top-right corner.
5
Click "Load unpacked" and select the unzipped folder.
6
You should see the SID extension appear. Pin it to your toolbar for easy access.
When an update is available, just download the new ZIP, replace the folder, and click the refresh icon on the extensions page.
